# Лабораторная 1
## Запуск на macOS

Ничего собирать не нужно.

1. Перейдите в папку `lab1`.
2. Откройте файл `binaries/CipherApp.dmg`.
3. Перетащите приложение в `Applications`.
4. Запустите приложение.

## Запуск на Windows 10-11

Ничего собирать не нужно.

1. Скопируйте папку `lab1` на Windows.
2. Откройте `lab1/binaries/CipherApp.exe`.
3. Запустите файл двойным кликом.
